\doxysubsection{Detailed Description}
\mbox{\hyperlink{classInterface_1_1Runner}{Runner}} asks players\textquotesingle{} for actions and executes them. (The actions, not the players \+:D.) 

Usage\+:
\begin{DoxyEnumerate}
\item Set control objects for the players.
\item Call \mbox{\hyperlink{classInterface_1_1Runner_a6ecbcede8d2ae8487abe26c2d57e9e44}{run()}}, which starts communicating with control objects to ask for actions which are then executed. For each action executed, action\+Performed-\/signal is emitted.
\item If \mbox{\hyperlink{classInterface_1_1Runner_a6ecbcede8d2ae8487abe26c2d57e9e44}{run()}} returns a pointer to a control object, the player managed by the object is in turn but the object couldn\textquotesingle{}t give the next move. Fix the state of the object or exchange it to a new one, and return to step 2.
\item If \mbox{\hyperlink{classInterface_1_1Runner_a6ecbcede8d2ae8487abe26c2d57e9e44}{run()}} returns a nullptr, game has ended.
\end{DoxyEnumerate}

If class method is not nothrow, \textquotesingle{}std\+::bad\+\_\+alloc\textquotesingle{} can always leak. (out of memory)
