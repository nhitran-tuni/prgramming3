\doxysubsection{Detailed Description}
defines the interface for agent cards. 

\mbox{\hyperlink{classInterface_1_1AgentInterface}{Agent\+Interface}} inherits from \mbox{\hyperlink{classInterface_1_1CardInterface}{Card\+Interface}}, meaning that to implement this interface, one must also implement cardinterface.

If class method is not nothrow, it can leak \textquotesingle{}std\+:bad\+\_\+alloc\textquotesingle{} ( out of memory ). \begin{DoxyInvariant}{Invariant}
if placement.\+lock() != nullptr, then \mbox{\hyperlink{classInterface_1_1AgentInterface_acc042fa81667ccce65b60f880b216516}{connections()}} $>$ 0 . In other words\+: if agent is on a location on board, it has connections. 
\end{DoxyInvariant}
