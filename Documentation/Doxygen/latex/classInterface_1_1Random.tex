\doxysubsection{Detailed Description}
\mbox{\hyperlink{classInterface_1_1Random}{Random}} is a singleton class for random number generation. 

All random numbers used in the game should be generated through this class, so that the numbers won\textquotesingle{}t become predictable.

If class method is not nothrow, \textquotesingle{}std\+::bad\+\_\+alloc\textquotesingle{} can always leak. (Out of memory)
