\doxysubsection{Detailed Description}
\mbox{\hyperlink{classInterface_1_1SettingsReader}{Settings\+Reader}} is a singleton class for reading settings. 

Through this class it is possible to read key-\/value pairs that are strings. The file can be set, and the setting are read lazily when foirst asked for or when asked specifically.

If class method is not nothrow, \textquotesingle{}std\+::bad\+\_\+alloc\textquotesingle{} can always leak. (Out of memory)
