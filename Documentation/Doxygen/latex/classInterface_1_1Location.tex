\doxysubsection{Detailed Description}
\mbox{\hyperlink{classInterface_1_1Location}{Location}} represents an area on the gameboard with related cards and tokens. 

All class methods have the precondition that the object is owned by shared\+\_\+ptrs.

All class methods, except \mbox{\hyperlink{classInterface_1_1Location_a4444c6e6220fe59651e8c4bd77b10493}{initialize()}}, have the precondition that \mbox{\hyperlink{classInterface_1_1Location_a4444c6e6220fe59651e8c4bd77b10493}{initialize()}}-\/method has been called.

If a method is not nothrow, \textquotesingle{}std\+::bad\+\_\+alloc\textquotesingle{} may always leak. (Out of memory)
