\doxysubsection{Detailed Description}
\mbox{\hyperlink{classInterface_1_1Player}{Player}} represents a player. 

Class defines player as a part of game state, not as a player who does actions.

All class methods have the precondition that this object is owned by shared\+\_\+ptr.

If class method is not nothrow, \textquotesingle{}std\+::bad\+\_\+alloc\textquotesingle{} can always leak. (out of memory)
