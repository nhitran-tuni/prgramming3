\doxysubsection{Detailed Description}
\mbox{\hyperlink{classInterface_1_1Game}{Game}} saves the static state. 

The game elements are meant to be set before starting the game. This class keeps track of the player in turn and if the game is on, but it doesn\textquotesingle{}t change the turns or end the game.

All class methods have the precondition that the object is owned by shared\+\_\+ptrs.

If method\textquotesingle{}s exception guarantee isn\textquotesingle{}t nothrow, method can always leak \textquotesingle{}std\+::bad\+\_\+alloc\textquotesingle{} (out of memory).
