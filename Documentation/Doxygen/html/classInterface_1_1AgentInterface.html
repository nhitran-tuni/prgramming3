Detailed Description

defines the interface for agent cards.

AgentInterface inherits from CardInterface, meaning that to implement this interface, one must also implement cardinterface.

If class method is not nothrow, it can leak 'std:bad_alloc' ( out of memory ).

Invariant
if placement.lock() != nullptr, then connections() > 0 . In other words: if agent is on a location on board, it has connections.
