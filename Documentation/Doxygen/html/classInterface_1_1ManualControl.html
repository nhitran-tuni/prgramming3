Detailed Description

ManualControl kuvaa käyttöliittymän kautta pelia pelaavaa ihmistä. Perii ControlInterface eli sisältää myös sen.

Tämä luokka on tarkoitettu kuvaamaan ihmispelaajaa, eikä osaa itsessään tuottaa siirtoja. Sen sijaan sille voi ulkoa päin (käyttöliittymästä käskien) asettaa siirron, jonka se sitten palauttaa seuraavaksi.

Jos luokan metodi ei ole poikkeustakuultaan nothrow, voi metodista aina vuotaa ulos poikkeus std::bad_alloc (muisti loppui).
