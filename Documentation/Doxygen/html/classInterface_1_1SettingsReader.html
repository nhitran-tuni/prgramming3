Detailed Description

SettingsReader is a singleton class for reading settings.

Through this class it is possible to read key-value pairs that are strings. The file can be set, and the setting are read lazily when foirst asked for or when asked specifically.

If class method is not nothrow, 'std::bad_alloc' can always leak. (Out of memory)
