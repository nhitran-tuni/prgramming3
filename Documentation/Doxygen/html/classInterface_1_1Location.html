Detailed Description

Location represents an area on the gameboard with related cards and tokens.

All class methods have the precondition that the object is owned by shared_ptrs.

All class methods, except initialize(), have the precondition that initialize()-method has been called.

If a method is not nothrow, 'std::bad_alloc' may always leak. (Out of memory)
