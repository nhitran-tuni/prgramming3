Detailed Description

Game saves the static state.

The game elements are meant to be set before starting the game. This class keeps track of the player in turn and if the game is on, but it doesn't change the turns or end the game.

All class methods have the precondition that the object is owned by shared_ptrs.

If method's exception guarantee isn't nothrow, method can always leak 'std::bad_alloc' (out of memory).
